# 🏡 NestEdge - MERN Estate Marketplace

A modern and responsive **Real Estate Marketplace** built with the **MERN Stack** that allows users to buy, sell, rent, and explore properties with an intuitive user experience. The application features secure authentication, property listing management, advanced search & filtering, and a clean responsive interface.

---

## 🚀 Features

- 🔐 JWT Authentication & Authorization
- 👤 User Registration & Login
- 🏠 Create, Update & Delete Property Listings
- 📸 Upload Property Images
- ❤️ Save Favorite Properties
- 🔍 Advanced Search & Filtering
- 📍 Browse Properties by Type & Location
- 📱 Fully Responsive UI
- ⚡ Fast and Modern React Interface
- 🛡 Secure REST APIs
- 🌐 MongoDB Database Integration

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/NestEdge-MERN-ESTATE.git
```

### Navigate into Project

```bash
cd NestEdge-MERN-ESTATE
```

### Install Backend Dependencies

```bash
cd api
npm install
```

### Install Frontend Dependencies

```bash
cd ../client
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
MONGO_URL=Your_MongoDB_URI

JWT_SECRET=Your_JWT_Secret

PORT=3000
```

---

# ▶️ Run the Application

### Backend

```bash
cd api
npm run dev
```

### Frontend

```bash
cd client
npm start
```
